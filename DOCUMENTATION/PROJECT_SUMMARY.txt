================================================================================
                          IMPORTANT NOTES
================================================================================

1. Database Setup:
   - Must run DATABASE_MIGRATION_SCHEMA.sql before using app
   - Use Supabase SQL Editor or psql command
   - Verify with verification queries in MIGRATION_GUIDE.md

2. Environment Variables:
   - Create .env file with Supabase credentials
   - Never commit .env to version control
   - Keep keys private and secure

3. Authentication:
   - Users must create account before using app
   - Auth context handles session management
   - Automatic user_settings creation

4. RLS Security:
   - All data is row-level secured
   - Users can only access own data
   - Policies are restrictive by default

5. Development:
   - Use npm run dev for development
   - Use npm run build for production
   - Check browser console for errors

6. Deployment:
   - Build before deploying
   - Update environment variables
   - Test in staging first
